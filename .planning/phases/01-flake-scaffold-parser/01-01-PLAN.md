**lib/default.nix** — the fromMiseToml function. It takes `{ lib }` as its module argument (from flake.nix) and exposes `fromMiseToml`. The function signature is `path: { pkgs }: derivation`. It must:

1. Read the file: `builtins.readFile path`
2. Parse TOML: `builtins.fromTOML`
3. Extract sections with guards: `config.tools or {}` and `config.env or {}`
4. Return `pkgs.mkShell { packages = []; }` (empty shell for Phase 1)

Exact content for lib/default.nix:

```nix
{ lib }:
{
  fromMiseToml = path: { pkgs }:
    let
      config = builtins.fromTOML (builtins.readFile path);
      tools  = config.tools or {};
      env    = config.env   or {};
    in
      pkgs.mkShell {
        packages = [];
      };
}
```

The `lib` argument is accepted but not used in Phase 1 — it sets up the pattern for later phases that will use `lib.mapAttrs`, `lib.filterAttrs`, etc.

The `tools` and `env` bindings are extracted but not used yet — Phase 2 maps tools to packages, Phase 4 maps env to shellHook.

Create the `lib/` directory if it does not exist.

Exact content for flake.nix:

```nix
{
  description = "mise2nix — converts mise.toml to a Nix devShell";

  inputs.nixpkgs.url = "github:NixOS/nixpkgs/nixpkgs-unstable";

  outputs = { self, nixpkgs }:
    let
      lib = nixpkgs.lib;
      forAllSystems = lib.genAttrs [
        "x86_64-linux"
        "aarch64-linux"
        "x86_64-darwin"
        "aarch64-darwin"
      ];
    in {
      lib = import ./lib { inherit lib; };

      devShells = forAllSystems (system:
        let pkgs = nixpkgs.legacyPackages.${system};
        in {
          default = self.lib.fromMiseToml ./mise.toml { inherit pkgs; };
        }
      );
    };
}
```
