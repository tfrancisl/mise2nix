   The `builtins.toString version` handles TOML bare integers (Pitfall 4).

3. Filter nulls and convert to list:


4. Replace `packages = [];` with `packages = resolvedPackages;`

The complete updated lib/default.nix should be:
```nix
{ lib }:
{
  fromMiseToml = path: { pkgs }:
    let
      runtimes = import ./runtimes.nix { inherit lib pkgs; };
      config   = builtins.fromTOML (builtins.readFile path);
      tools    = config.tools or {};
      env      = config.env   or {};

      resolvedOrNull = builtins.mapAttrs (name: version:
        if runtimes ? ${name}
        then runtimes.${name} (builtins.toString version)
        else null
      ) tools;

      resolvedPackages = builtins.filter (v: v != null)
                           (builtins.attrValues resolvedOrNull);
    in
      pkgs.mkShell {
        packages = resolvedPackages;
      };
}
```

IMPORTANT: Do NOT add `builtins.toString` inside runtimes.nix resolver calls AND in default.nix -- apply it in exactly one place. The action above applies it in default.nix at the call site (consistent location for all tool resolution, including future utility tier in Phase 3).
