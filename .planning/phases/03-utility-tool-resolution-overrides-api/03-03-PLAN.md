**Check 4: unknown-tool-error**
Verifies that an unknown tool with no override throws a descriptive error.

```nix
unknown-tool-error =
  let
    toml = builtins.toFile "unknown-test.toml" ''
      [tools]
      nonexistent_tool_xyz = "latest"
    '';
    # Try to evaluate and catch the expected error
    result = builtins.tryEval (self.lib.fromMiseToml toml { inherit pkgs; });
  in pkgs.runCommand "unknown-tool-error" {} ''
    ${if result.success then
      ''echo "FAIL: should have thrown for unknown tool" && exit 1''
    else
      ''echo "PASS: unknown tool correctly throws error" > $out''
    }
  '';
```

IMPORTANT: The `builtins.tryEval` approach works for `builtins.throw` -- it catches the error and sets `result.success = false`. This is the standard Nix pattern for testing error paths in check derivations.
