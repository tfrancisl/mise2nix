**Check 3: `wrapper-use-writes-toml`** (WRAP-02)
Verifies that `mise use "pipx:black"` writes `"pipx:black" = "latest"` to mise.toml.

```nix
wrapper-use-writes-toml = let
  miseWrapper = pkgs.writeShellScriptBin "mise" ''
    ... (EXACT same wrapper script as in lib/default.nix) ...
  '';
  tomlFixture = builtins.toFile "fixture.toml" ''
    [tools]
    node = "22"
  '';
in
  pkgs.runCommand "wrapper-use-writes-toml"
    {nativeBuildInputs = [miseWrapper pkgs.gnused pkgs.grep];}
    ''
      cp ${tomlFixture} mise.toml
      chmod +w mise.toml
      MISE_CONFIG_FILE=mise.toml mise use "pipx:black"
      if ${pkgs.grep}/bin/grep -q '"pipx:black" = "latest"' mise.toml; then
        echo "PASS: entry written to mise.toml" > $out
      else
        echo "FAIL: entry not found in mise.toml"
        cat mise.toml
        exit 1
      fi
    '';
```

**Check 4: `wrapper-use-prints-message`** (DX-05)
Verifies that `mise use` output contains `[mise2nix]` attribution.

```nix
wrapper-use-prints-message = let
  miseWrapper = pkgs.writeShellScriptBin "mise" ''
    ... (EXACT same wrapper script as in lib/default.nix) ...
  '';
  tomlFixture = builtins.toFile "fixture.toml" ''
    [tools]
  '';
in
  pkgs.runCommand "wrapper-use-prints-message"
    {nativeBuildInputs = [miseWrapper pkgs.gnused pkgs.grep];}
    ''
      cp ${tomlFixture} mise.toml
      chmod +w mise.toml
      MISE_CONFIG_FILE=mise.toml mise use "npm:prettier" > output.txt 2>&1
      if ${pkgs.grep}/bin/grep -q "mise2nix" output.txt; then
        echo "PASS: mise2nix attribution in output" > $out
      else
        echo "FAIL: no mise2nix attribution"
        cat output.txt
        exit 1
      fi
    '';
```

IMPORTANT: The wrapper script in each check MUST be copied verbatim from lib/default.nix. Read the file first. Do not paraphrase or abbreviate.

IMPORTANT: The tomlFixture files created by `builtins.toFile` are read-only in the Nix store. The check MUST `cp` the fixture and `chmod +w` before running the wrapper (which uses `sed -i` to modify in-place).

IMPORTANT: Use `${pkgs.grep}/bin/grep` in the check body for consistency (though PATH-based grep would work inside runCommand since nativeBuildInputs sets PATH).
