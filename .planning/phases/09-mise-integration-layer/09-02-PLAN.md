# Plan 09-02: shellHook + README + nix checks

## Goal
Auto-activate mise in the devShell and document shell activation for fish/zsh/nu users.

## Changes

### lib/default.nix — shellHook
Add `shellHook` to the mkShell call:
```bash
eval "$(${pkgs.mise}/bin/mise activate bash)"
```
shellHook always runs as bash; this installs PROMPT_COMMAND for `nix develop` users.
direnv users get MISE_INSTALLS_DIR exported automatically; other shells need one-time rc setup.

### README.md — Shell Activation section
Document one-time setup for bash/zsh/fish/nushell users.

### flake.nix — new check
Add `mise-installs-dir` check verifying:
- MISE_INSTALLS_DIR is set
- installs dir contains expected tool structure (core: bin/ subdir; aqua: binary directly)
